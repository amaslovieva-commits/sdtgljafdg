## Описание программы
*"Expense Tracker (Трекер расходов) — это консольное приложение для отслеживания личных расходов с фильтрацией, подсчётом суммы за период, JSON и Git.**
---
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`
